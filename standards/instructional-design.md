# Instructional Design Best Practices

> **Plugin Reference** — This document is part of the ASU Canvas Builder plugin.
> A framework-agnostic collection of evidence-based instructional design principles.
> Referenced by: `course-build`, `syllabus-generator`, and all content generation skills.

## 1. Backward Design / Understanding by Design (UbD)
- **Framework**: Start with desired results → determine acceptable evidence → plan learning experiences
- **Stage 1**: Identify enduring understandings and essential questions per module
- **Stage 2**: Design performance tasks and rubrics that reveal understanding
- **Stage 3**: Plan instruction that enables students to demonstrate understanding
- **Source**: Wiggins & McTighe (2005) — Understanding by Design (2nd ed.)
- **Application**: Each module starts from learning objectives → assessment → then content

## 2. ADDIE Model
- **Phases**: Analysis → Design → Development → Implementation → Evaluation
- **Application**: Analysis/Design phases produce the course blueprint; Development is the Canvas build
- **Key**: Iterative — evaluation feeds back into analysis for continuous improvement

## 3. Bloom's Taxonomy for Learning Objectives
- **Levels**: Remember → Understand → Apply → Analyze → Evaluate → Create
- **ASU Standard**: 3-5 objectives per module using measurable Bloom's verbs
- **Verb examples**: Explain (understand), Predict (apply), Compare (analyze), Evaluate (evaluate)
- **Tip**: Match Bloom's level to course level — introductory courses emphasize Understand/Apply; advanced courses emphasize Analyze/Evaluate/Create

## 4. Mayer's 12 Multimedia Learning Principles
| Principle | Rule |
|---|---|
| Coherence | Remove extraneous material |
| Signaling | Highlight essential material |
| Redundancy | Don't duplicate narration as on-screen text |
| Spatial Contiguity | Place text near corresponding graphics |
| Temporal Contiguity | Present narration and animation simultaneously |
| Segmenting | Break complex lessons into learner-paced segments |
| Pre-training | Teach key terms/concepts before main lesson |
| Modality | Use narration rather than on-screen text with graphics |
| Multimedia | Use words and pictures, not words alone |
| Personalization | Use conversational tone |
| Voice | Use human voice, not machine |
| Image | Speaker image not necessarily needed |

- **Source**: Mayer (2021) — Multimedia Learning (3rd ed.), Cambridge University Press

## 5. Universal Design for Learning (UDL)
- **Three principles**: Multiple means of engagement, representation, action/expression
- **Plus-One approach**: Add one alternative for each content type (caption videos, alt text images, transcript for audio)
- **ASU requirement**: WCAG 2.1 Level AA compliance by April 24, 2026
- **FRAME alignment**: ASU's FRAME (Fonts, Readability, Accessible Media, Meaningful Hyperlinks, Equitable Design) framework operationalizes accessibility — see `standards/canvas-standards.md` for full FRAME specification
- **Inclusive language**: Use asset-based framing, diverse examples, and welcoming tone across all course content
- **Application**: Every module should offer content in multiple formats

## 6. Quality Matters (QM) Higher Education Rubric (7th Ed.)
- **8 General Standards**: Course overview, learning objectives, assessment, instructional materials, learning activities, course technology, learner support, accessibility/usability
- **Key annotations**: Alignment between objectives → assessments → activities → materials
- **Application**: Use QM standards as quality checklist during development

## 7. Video Design Best Practices
- **Optimal length**: 6 minutes (MIT/edX study by Guo, Kim & Rubin, 2014)
- **Structure**: Hook → Objective → Content → Check → Summary → Bridge
- **Engagement**: Talking head + tablet drawing outperforms slides (Guo et al.)
- **Production**: Khan-style drawing with narration highly effective
- **For Canvas courses**: Audio primers (~5 min casual overview) + full lecture videos per module
- **Source**: Guo, P.J., Kim, J., & Rubin, R. (2014). "How video production decisions affect student engagement." L@S Conference.

## 8. Active Learning & Retrieval Practice
- **Retrieval practice**: Testing enhances long-term retention more than re-studying
- **Spacing effect**: Distribute practice across time rather than massing
- **Interleaving**: Mix problem types within practice sessions
- **Application**: Knowledge checks after lessons, spaced quizzes, varied question types
- **Sources**: Roediger & Butler (2011); Dunlosky et al. (2013)

## 9. Scaffolding & Cognitive Load Theory
- **Intrinsic load**: Complexity inherent in content (discipline-dependent)
- **Extraneous load**: Poor design adds unnecessary difficulty (REDUCE this)
- **Germane load**: Effort that contributes to learning (INCREASE this)
- **Scaffolding approaches**: Worked examples → faded examples → independent practice
- **Application**: Start with guided practice, gradually release responsibility
- **Source**: Sweller, Ayres, & Kalyuga (2011) — Cognitive Load Theory

## 10. Formative Assessment Strategies
- **Low-stakes frequent checks**: Knowledge checks every module (ASU gold standard)
- **Immediate feedback**: Auto-graded quizzes with explanatory feedback
- **Self-assessment**: Reflection prompts at module conclusions
- **Peer learning**: Discussion-based assessments where students explain to each other
- **Application**: 5-min knowledge check + guided practice per module
